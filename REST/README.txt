
Build a REST API with Node.js and Express.js
What will I learn?

    * designing REST APIs with Node.js and Express.js
    * organizing routes for large scale apps
    * hierarchical app configuration with files and environment variables
    * user authentication with Passport.js
    * data modelling
    * data storage using MongoDB and Mongoose

What will I build?

In this workshop you will build the backend of a Twitter web app clone.

Your app will implement features like user creation and authentication, ability to post tweets, 
follow/unfollow users or build streams of tweets based on different criteria.
